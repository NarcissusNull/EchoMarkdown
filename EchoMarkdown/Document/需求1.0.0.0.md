#需求分析
##功能需求
+ 支持中英文输入
+ 支持Markdown基本语法
+ 支持实时预览
+ 支持语法高亮*
+ 支持新建，打开，保存，另存为操作
+ 支持查找与替换*
+ 支持字数统计和行列显示*
+ 支持显示行号*
+ 支持转码为PDF，HTML等格式文件*
+ 支持更改实时预览效果（更换css）
+ 支持未保存提示和自动保存*
+ 支持更改编辑字体及大小，颜色*
+ 支持撤销和重做功能*
+ 支持复制，剪切和粘贴
+ 支持检查更新*
+ 支持下载源码*
+ 支持更换皮肤*
+ 支持配置文件

##性能需求
+ 本机运行不卡顿
+ 启动不卡顿
+ 实时预览同步刷新
+ 后台转码不影响使用
+ 退出不卡顿
+ 允许编辑20M以内的文件
